1. 项目完成情况

3. 布局
界面布局和示例图片布局相同
4. 图片排版问题
首页的图片并没有使用square文件夹下的图片, 而是将图片以display: flex;的方式进行展示, 每个图片设置好宽度百分比, 从而实现图片的缩放
其他页面图片使用的是经过裁剪的方形图片.
